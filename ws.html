🌐 WebSocket Bridge

Node.js webserial-core v2
Requires the Node.js bridge server running locally:
cd demos/websocket && node server.js

The bridge relays serial I/O over WebSocket JSON messages so the browser can talk to any serial port on the server machine — including behind firewalls or on embedded Linux devices.
← Web Serial (native) ← WebUSB Polyfill ← Web Bluetooth (BLE UART)
