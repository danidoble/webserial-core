🔌 WebUSB Polyfill

WebUSB webserial-core v2
Uses WebUsbProvider to bypass the native Web Serial API. The same AbstractSerialDevice API communicates directly with USB endpoints — useful on Android Chrome where Web Serial is unavailable.

Adjust the usbVendorId / usbProductId filter in main.ts to match your hardware.
← Web Serial (native) → Web Bluetooth (BLE UART) → WebSocket Bridge (Node.js)
