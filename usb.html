🔌 WebUSB Polyfill Test

POLYFILL WebSerial Core v2
⚠ Desktop limitation: On Linux, Windows, and macOS, USB-serial devices (CP2102, CH340, FTDI, etc.) are automatically claimed by OS kernel drivers (cp210x, ch341, usbserial). WebUSB cannot access devices already claimed by a driver — the device picker will appear empty.

This polyfill is designed for Android Chrome, where the OS does not load USB-serial kernel drivers, leaving the device available for WebUSB access.

On desktop, use the Native Web Serial test instead.
