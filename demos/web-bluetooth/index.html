📡 Web Bluetooth

BLE GATT webserial-core v2
Uses WebBluetoothProvider with the Nordic UART Service (NUS) over GATT. Compatible devices: nRF52 series, ESP32 BLE UART, or any module advertising the NUS service UUID 6e400001-….

Requires HTTPS or localhost and a Chromium-based browser with Web Bluetooth enabled.
